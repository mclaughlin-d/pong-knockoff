# pong-knockoff
A version of the classic 'Pong' game, created in Java 15 in 2021.  

## Instructions
Press 'n' to start a new game, 's' to serve the ball, and 'q' to quit the current game. Information about the game (score, if game was ended, etc is displayed in the bottom left corner of the screen).
